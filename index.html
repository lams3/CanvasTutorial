HTML Canvas API

O <canvas> é um elemento HTML que nos permite trabalhar com gráficos usando Javascript. Iremos abordar as funções básicas de sua API, bem como mostrar alguns exemplos interativos.

Considerações Iniciais

O primeiro passo para a utilização do canvas é a criação de uma página HTML contendo um canvas, assim como sua seleção por meio do código Javascript e a criação de um contexto de desenho para o mesmo. O canvas pode ser selecionado como qualquer outro objeto do DOM.

Cada elemento <canvas> tem atributos widht e height que definem seu tamanho em pixels. Esses atributos são diferentes dos atributos CSS de mesmo nome. Ao modificar o CSS o canvas tem sua imagem e sistema de coordenadas distorcido para se ajustar aos valores especificados, mas seus atributos width e height não mudam. Podemos modificar esses atributos de forma correta usando a tag HTML, ou via Javascript.

O canvas tem sua origem inicialmente fixada em seu canto superior esquerdo, com o eixo x crescendo à direita, e o y abaixo, onde suas coordenadas são dadas de (0, 0) a (width, height).

Paths e Formas Básicas

Podemos imaginar o desenho no canvas como o desenho de uma caneta no papel. Cada caneta tem propriedades como cor e espessura do traço. Podemos imaginar um Path como um desenho feito por uma única caneta. Cada path tem propriedades como cor de preenchimento, cor do traço, espessura do traço, dentre outras. Sempre que quisermos 'mudar de caneta' é recomendado que criemos um novo Path.

Algumas funções de Paths do canvas são:

  • beginPath()
  • moveTo(x, y)
  • lineTo(x, y)
  • arcTo(x1, y1, x2, y2, raio)
  • arc(xc, yc, raio, inicio, fim [, sentido])
  • rect(x, y, largura, altura)
  • clearRect(x, y, largura, altura)
  • stroke()
  • fill()

E algumas propriedades:

  • strokeStyle
  • fillStyle
  • lineWidth

Experimente!

Interagindo com o Usuário

É possivel criar formas de interação no canvas usando eventos Javascript. Se quisermos por exemplo criar um circulo que pode ser arrastado pelo canvas, podemos dividir esse evento em três partes básicas. Apertar botão, arrastar mouse e soltar botão.

Criando Animações

Podemos também criar animações no canvas usando Javascript. Uma das maneiras é com a função setInterval(função, intervalo). Suponhamos que agora queremos que o circulo ocile de um lado a outro da tela.
